# Expense Sharing Assignment

This project is a Java backend implementation of an Expense Sharing Application (Splitwise-like).

## Features
- Create groups and add members
- Add shared expenses with:
  - Equal split
  - Exact amount split
  - Percentage split
- Track balances between users
- Simplify balances to minimize transactions
- Settle dues between users
- User-wise balance view

## How to Run
1. Open the project in Eclipse or any Java IDE
2. Run `MainApp.java`
3. Output will show simplified balances, user-wise balances, and settlements
